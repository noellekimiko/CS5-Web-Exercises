Extra Credit Lab 2: HTML and CSS on your own webserver...

In the last exercise, we worked with HTML to make a simple web page. HTML (or Hyper-Text Markup Language) is only part of the source code for most web pages. It controls the content that goes on the page. When CSS (Cascading Style Sheets) is used, we can control how that content is displayed; it is pure styling. You can completely change the appearance of a web page by just changing the CSS, without manipulating the HTML at all. Here are two thorough references for these technologies -- they may be a bit too thorough for our intents and purposes, but they're still good to have around:

  • HTML: http://w3schools.com/html/default.asp
  • CSS: http://w3schools.com/css/default.asp, http://www.csszengarden.com/

For this problem we'd like to give you a chance to practice your HTML and CSS skills while also telling us a little more about yourselves. You should prepare a personal page with a photo of you (or of something you love) and with information like your name, your hometown, your favorite things (food, color, sport, etc...), and anything else that you think is interesting (or that you just feel like telling us!).

To start, download this zip file named htdocs.zip. Then

  • Unzip those files
  • Copy all of the files (but not the folder) to your web directory


Then open the CGI server using the python -m CGIHTTPServer command as we did in Extra Credit Lab 1. Let's first take a look at index2.html. To view it, go to the following address in your browser window:

http://localhost:8000/index2.html

You should see the second of the two screenshots at the bottom of this page.

Using a text editor (Sublime will work just fine), change the line of the index2.html file that reads 

<link rel="stylesheet" type="text/css" href="style2.css" />

to a line that reads 

<link rel="stylesheet" type="text/css" href="style1.css" />

and reload the page to see the other version: the same content, but with a different style sheet.


What to do

Alter the provided index.html or index2.html webpage to create your own webpage using HTML and CSS. Also change at least one of the two CSS files (style1.css or style2.css). You should be sure to change each of these things somewhere in your HTML and CSS. Each one is worth 10 points:

  • The color of your text. This link explains HTML colors -- try the rgb variety (I find that easier.)
  • The color of your links. (Use the same reference as above, but see this link to explain changing links in CSS.)
  • The image(s) you display: choose your own, download it, copy it to your web directory, and use it! (See Extra Credit Lab 1 if you need a reminder.)
  • The color or content of your background. This link explains CSS backgrounds -- try using an image!
  • Finally, experiment with at least one HTML or CSS feature of your choice. For example, although it's not considered stylish, you might use Marquee text!


Finally, load into your browser your personal page with each of those two style sheets: the one you changed and the one you (perhaps) did not change.

For your page and each of the two stylesheets, take a screenshot and name your screenshots

  • hw2pr1img1.png
  • hw2pr1img2.png
If you can make sure they're .png (Portable Network Graphics) files, great! But if they're another kind of image file, that's OK, too.


Submission

For this assignment, you should not submit your HTML or CSS files, just the two images, which are your screenshots.
